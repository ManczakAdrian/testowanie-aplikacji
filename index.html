Instalacja Cypress

            
                npm install cypress --save-dev

Dodatnie skryptu do projektu

            
                {
                    "scripts": {
                      "cypress:open": "cypress open"
                    }
                  }

Uruchomienie serwera Cypresowego

            
                npm run cypress:open

Mała przypadłość :/

            
                npm install -D eslint-plugin-cypress

                "eslintConfig": { "extends": [ "plugin:cypress/recommended" ] },
